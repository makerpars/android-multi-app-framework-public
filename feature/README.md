# Feature Modules

This directory contains the feature-specific modules of the application. Each module represents a distinct user-facing feature or screen.

## Modules
- **content**: Displays the main text/media content.
- **audio**: Handles audio playback, services, and notification controls.
- **ads**: Ad loading, presentation, and ad-related UI components.
- **billing**: Subscription and in-app billing flows.
- **auth**: Feature-level authentication screens and flows.
- **notifications**: Notification list and related interactions.
- **messages**: User message list and send/reply interactions.
- **settings**: User preferences and app configuration.
- **otherapps**: Cross-promotion and other-apps listing screens.

## Navigation
Features are modularized and integrated by the app-level navigation graph, with selected deep-link entry points.
