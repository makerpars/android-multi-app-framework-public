# Core Modules

This directory contains the foundational modules of the application. These modules are designed to be reusable and independent of feature logic.

## Modules
- **common**: General purpose extensions and utilities.
- **model**: Domain models and data classes.
- **database**: Room database definitions, DAOs, and entities.
- **datastore**: Proto DataStore implementation for preferences.
- **designsystem**: Compose UI components, theme, and typography.
- **firebase**: Firebase wrappers and integrations (Analytics, Crashlytics, Auth, Firestore, Messaging, Remote Config, Performance, App Check).
- **auth**: Authentication-related abstractions and helpers used by feature modules.
